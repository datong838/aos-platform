- `Amazon电商接入方案/00-Amazon-AOS对接方案.md`
- 10 域 + ASIN Catalog，10 Object Types + 10 Link Types
- 关键发现：**LWAAuth + STS Token 双令牌**（1h 轮换）、AWS Signature V4、FBA vs MFN 双轨物流、ASIN + Seller SKU 双标识、三区域（NA/EU/FE）独立端点、Reports + Feeds 批量通道
